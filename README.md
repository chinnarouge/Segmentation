# Tumor Segmentation Project

This repository contains scripts for tumor segmentation using a 3D U-Net model. The project involves data extraction, visualization, and segmentation of medical images. The pipeline includes data preprocessing, model training, and evaluation.

---

## Project Structure

- **`requirements.txt`**: Lists all dependencies required to run the project.
- **`extraction.py`**: Script for extracting and preprocessing medical image data.
- **`view_images.py`**: Script for loading and visualizing medical images and their segmentations.
- **`segmentation.py`**: Contains the 3D U-Net model, training pipeline, and evaluation scripts.

---
